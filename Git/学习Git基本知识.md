### 3.git的基本流程
```向仓库中添加文件流程
1.我们先在工作区中编写好代码
2.把编写好的代码添加到暂存区，让后面有一个回旋的余地，可以修改代码，删除代码；(用 git add . / git add test.php 命令，该命令是把代码从工作区添加到暂存区)
3.最后把代码文件提交到仓库，成为一个新的版本，并且对他人可见；（把代码从暂存区提交到仓库，用git commit -m "提交描述"）
```
